<section>
  <h2>📌 Overview</h2>
  <p>
    Local Fixperts is a full-stack service marketplace platform that connects users
    with skilled local professionals like electricians, plumbers, and carpenters.
  </p>
  <p>
    Built using the MERN stack, it provides a seamless way to browse, book,
    and manage services online.
  </p>
</section>

<section>
  <h2>🚀 Features</h2>

  <h3>👤 User Features</h3>
  <ul>
    <li>User Registration & Login</li>
    <li>Browse services by category</li>
    <li>Search experts</li>
    <li>Book services</li>
    <li>Track booking status</li>
    <li>Ratings & Reviews</li>
  </ul>

  <h3>🧑‍🔧 Expert Features</h3>
  <ul>
    <li>Expert registration</li>
    <li>Profile management</li>
    <li>Manage service requests</li>
    <li>Track jobs</li>
  </ul>

  <h3>⚙️ System Features</h3>
  <ul>
    <li>Secure authentication</li>
    <li>Responsive UI</li>
    <li>Real-time updates</li>
    <li>Efficient data management</li>
  </ul>
</section>

  <h2>🔄 Workflow</h2>
  <ol>
    <li>User registers or logs in</li>
    <li>Browses or searches services</li>
    <li>Views expert profiles</li>
    <li>Books a service</li>
    <li>Expert responds</li>
    <li>Service completed</li>
    <li>User gives feedback</li>
  </ol>

<section>
  <h2>⚡ Installation</h2>

  <h3>Clone Repository</h3>
  <code>git clone https://github.com/sandhuxvishav/Local-Fixpert</code>

  <h3>Install Dependencies</h3>
  <code>
cd backend && npm install  <br>
cd frontend && npm install
  </code>

  <h3>Run Project</h3>
  <code>Backend :<br>node index.js 
    <hr>Frontend :<br>npm run dev
  </code>
</section>
