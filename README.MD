<section>
  <h2>📌 Overview</h2>
  <p>
    Local Fixperts is a full-stack service marketplace platform that connects users
    with skilled local professionals like electricians, plumbers, and carpenters.
  </p>
  <p>
    Built using the MERN stack, it provides a seamless way to browse, book,
    and manage services online.
  </p>
</section>

<section>
  <h2>🚀 Features</h2>

    <li>Search experts</li>
    <li>Book services</li>
    <li>Track booking status</li>
    <li>Ratings & Reviews</li>
  </ul>
